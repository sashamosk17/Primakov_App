# PrimakovApp                                                                                                                                                          
                                                                                                                                                                         
  Кроссплатформенное мобильное приложение для автоматизации учебного процесса в Гимназии им. Примакова.                                                                  
                                                                                                                                                                         
  ## Описание                                                                                                                                                            

  PrimakovApp — цифровой помощник для учеников, учителей и администрации гимназии. Приложение предоставляет доступ к расписанию, дедлайнам, новостям, меню столовой и
  другим сервисам.

  ### Основные функции

  - Просмотр индивидуального расписания уроков с информацией о времени, предмете, кабинете и преподавателе
  - Создание и управление дедлайнами по учебным заданиям с напоминаниями
  - Просмотр Stories с актуальными новостями и мероприятиями гимназии
  - Анонимное оценивание преподавателей (доступно студентам)
  - Просмотр меню школьной столовой с информацией о калорийности и составе блюд
  - Просмотр объявлений гимназии
  - Редактирование профиля и смена пароля
  - Создание заявок на техническую поддержку, ремонт и уборку

  ## Технологический стек

  - **Backend**: Node.js + Express + TypeScript, Clean Architecture / DDD, PostgreSQL 14+
  - **Frontend**: Flutter 3.0+ (Riverpod, Dio, url_launcher, shared_preferences)
  - **Аутентификация**: JWT токены (срок действия 7 дней)

  ## Системные требования

  ### Для использования приложения

  - **Android**: версия 10.0 и выше
  - **iOS**: версия 14.0 и выше
  - 2 ГБ оперативной памяти (рекомендуется 4 ГБ)
  - 100 МБ свободного места
  - Подключение к интернету (Wi-Fi или мобильная сеть)

  ### Для разработки

  - Flutter SDK версии 3.41 или выше
  - Node.js версии 20.x
  - PostgreSQL версии 14+
  - Android Studio или Android SDK (для Android)
  - Xcode (для iOS, только на macOS)

  ## Установка

  ### Установка готового APK для Android

  1. Получите установочный файл `PrimakovApp.apk` у администратора гимназии
  2. Откройте файл на вашем устройстве
  3. Если появится предупреждение "Установка из неизвестных источников", разрешите установку:
     - Откройте Настройки → Безопасность → Неизвестные источники
     - Включите опцию для браузера или файлового менеджера
  4. Нажмите "Установить" и дождитесь завершения установки
  5. Нажмите "Открыть" или найдите иконку PrimakovApp на главном экране

  ### Сборка APK из исходного кода

  1. Откройте командную строку (терминал)

  2. Клонируйте репозиторий с исходным кодом:
     ```bash
     git clone https://github.com/sashamosk17/Primakov_App.git

  3. Перейдите в папку с проектом Flutter:
  cd Primakov_App/flutter
  4. Загрузите необходимые зависимости:
  flutter pub get
  5. Выполните сборку APK-файла, указав адрес сервера гимназии:
  flutter build apk --dart-define=API_BASE_URL=http://185.221.198.242:3000/api
  6. Готовый установочный файл будет создан по пути:
  build/app/outputs/flutter-apk/app-release.apk
  7. Скопируйте файл app-release.apk на мобильное устройство (через USB, облачное хранилище или мессенджер)
  8. Установите APK на устройство, следуя инструкциям выше (шаги 2–5)

  Примечание: При самостоятельной сборке убедитесь, что путь к проекту не содержит символов кириллицы и пробелов.

  Установка на iOS

  1. Откройте Xcode на компьютере Mac
  2. Скачайте или клонируйте репозиторий с исходным кодом:
  git clone https://github.com/sashamosk17/Primakov_App.git
  3. В Xcode выберите "Open a project or file" и откройте папку flutter/ios (файл Runner.xcworkspace)
  4. Подключите устройство iPhone/iPad к компьютеру через USB-кабель
  5. В Xcode выберите ваше устройство в верхней панели (справа от кнопки "Run")
  6. При необходимости настройте подпись кода (Signing & Capabilities):
    - В разделе "Team" выберите свою учетную запись Apple (бесплатная подойдет)
    - Измените Bundle Identifier на уникальный (например, com.yourname.PrimakovApp)
  7. Нажмите кнопку "Run" (треугольник) или нажмите Cmd + R
  8. Xcode соберет приложение и установит его на устройство
  9. На устройстве может появиться запрос о доверии разработчику:
    - Откройте Настройки → Основные → VPN и управление устройством
    - Нажмите на профиль разработчика и выберите "Доверять"
  10. Запустите приложение на устройстве
