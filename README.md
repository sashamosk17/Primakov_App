# PrimakovApp

Этот репозиторий содержит бэкенд (Node.js/Express + TypeScript) и мобильное приложение (React Native + Redux Toolkit), созданные на основе архитектурной документации.

- бэкенд: чистые архитектурные слои, доменная модель, вдохновленная DDD
- мобильное приложение: экраны, навигация, срезы Redux, хуки и сервисы

Подробности см. в папках backend и mobile.
